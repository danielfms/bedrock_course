# Create a virtual environment:
Windows:
```
py -m venv .venv
```
Unix:
```
python3 -m venv .venv
```

# Activate a virtual environment:
Windows:
```
.venv\Scripts\activate

Git bash: source .venv/Scripts/activate
```
Unix:
```
source .venv\bin\activate
```

# Install libraries
```
pip install -r .\requirements.txt
```
